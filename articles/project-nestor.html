Project Nestor

Introduction¶

Together with Bruno Kinoshita I am involved in a project for the development of a new testcase management application. The working title of the application is Nestor.

Project team¶

  • co-founder of Tupilabs
  • Apache committer
  • developer of the Testlink Jenkins plugin
  • developer of Testopia Jenkins plugin

Peter Florijn

  • Owner of Peter Florijn Consultancy B.V.
  • co-developer of the Testopia plugin for Jenkins.
  • contributor to Bugzilla project
  • contributor to Testopia project
