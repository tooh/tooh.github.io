Activate spellcheck in Sublime Text 3

By default spellcheck is off in Sublime Text.

{% docs note %} You can’t edit the default settings file. If you try this, you cannot edit the setting.

In order to activate add the following to the User settings:

,
    // Set to true to turn spell checking on by default
    "spell_check": true
