Installation notes homebridge

By Peter Florijn: - vr 11 oktober 2019 Updated on okt 11, 2019

Automation homebridge plugins

In this post I have documented how I have configured my homebridge setup on my raspberry pi 3.

This is a work in progress and the post will be updated when necessary

  • Services
    • systemd
    • start , stop en monitor
  • Installation plugins
    • config-ui-x
  • Homebridge config

Services¶

systemd¶

automatic restart

start , stop en monitor¶

$ sudo systemctl stop homebridge $ sudo systemctl start homebridge $ sudo journalctl -f -u homebridge

Installation plugins¶

config-ui-x¶

sudo npm --verbose install -g homebridge-config-ui-x --unsafe-perm
