Custom alfred search for jira

Sun 31 January 2016
By Peter Florijn
Category: articles
Tags: Jira Alfred workflow

Summary

Recently we switched from using Bugzilla for our issue tracking to Jira.

As an Alfred user I went looking for some Alfred support for Jira searches.

I found a couple of workflows for this, but for now I settled for a simple custom web search in Alfred.

Note

Replace [your jira server] with your jira server.

https://[your jira server]/secure/QuickSearch.jspa?searchString={query}
Proudly powered by Pelican, which takes great advantage of Python.
